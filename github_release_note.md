종목 추가 시 6자리 종목코드 입력 방식으로 변경했습니다.

- 한글 종목명 검색은 TOS 리스크로 인해 비활성화

- 비주식 종목(뮤추얼펀드 등)이 잘못 매칭되는 문제 수정
- 종목명 표시 시 `shortName`을 우선 사용하여 정확도 개선
